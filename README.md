# flutter-docker-template

flutter開発用のDockerテンプレート
fvmを利用して環境構築. バージョン等の情報は`.env`で管理.

## setup flutter project

下記コマンドを実施して作成されるファイルは配置済み.

```bash
fvm flutter create foo
```

## build

整備中

```bash
fvm flutter build web
```

## run

整備中

```bash
fvm flutter run -d chrome
```
